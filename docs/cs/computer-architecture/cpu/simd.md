# Single Instruction Multiple Data (SIMD)

Даёт возможность применить какую-то операцию к нескольким элементам разом, обычно очень сильно греют процессор

Как правило в процессоре для поддержки этих инструкций создают отдельные регистры и отдельные вычислительные блоки для работы с этими регистрами

Современные компиляторы часто применяют векторизацию (использование SIMD) автоматически

## Streaming SIMD Extensions (SSE)

Изначально для ускорения работы с плавающими числами

| Версия | Возможности |
|---|---|
| 1 | Восемь 128-битных регистров для 4 чисел по 32 бит (с плавающей точкой) |
| 2 | Теперь 2 64-битных числа в регистре |
| 3 | Уже 13 инструкций |
| 4.1 | 47 инструкций (ускорение видео) |
| 4.2 | 54 инструкции (операции со строками) |

Операции со строками добавили из-за популярности xml

## Advanced Vector Extensions (AVX)

| Версия | Возможности |
|---|---|
| AVX | 2011 год: 16 256-битных регистров |
| AVX2 | 2013 год: Fused Multiply-Add, «умножение-сложение с однократным округлением» |
| AVX-512 | 2016 год: Целочисленные FMA и алгоритмы свёрточной (конволюционной) нейронной сети (CNN-алгоритмы) |
| AVX-10 | 2023 год: ? |
