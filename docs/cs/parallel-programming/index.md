# Параллельное программирование

## Зачем нам параллелки?

Чтобы что-то быстро вычислить, например:

- Генетика и фолдинг белков 🧬
- Климатология ☂️
- Физика (симуляции) / Астрономия (обработка космо-снимков)
- Банковские транзакции / Такси / Карты
- HFT 📈

## Как работает?

Когда ядро одно: [SIMD](../computer-architecture/cpu/simd.md)

Когда ядер много: [потоки и процессы](process-vs-thread.md), OpenMP

Когда много машин: кластера машин (hadoop)
