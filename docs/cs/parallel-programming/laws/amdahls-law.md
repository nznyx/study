# Закон Амдала

Общее ускорение программы при распараллеливании процессов всегда ограничено той ее частью, которая должна выполняться последовательно

- P -- Доля последовательного кода (I/O, рекурренты, примитивы синхронизации)
- N -- число процессов

Тогда ускорение $S(N)$ вычисляется как:

$$
S(N) = \frac{1}{P + \frac{1 - P}{N}}
$$
