# Inter Process Communication (IPC)

Механизмы общения и передачи данных между различными процессами внутри OS

## Какие механизмы есть?

1. Файлы
2. `socket` - на нём основаны серверы и брокеры обмена сообщениями
3. `pipe`
4. `semophore` - не передают данные, дают возможность сказать свободен/занят ли ресурс
5. `shmem` - Shared Memory
6. [`signals`](../../os/signals/index.md) - не можем передать данные, это о передаче "событие произошло"
7. Очереди ядра (в зависимости от OS)
8. итд
