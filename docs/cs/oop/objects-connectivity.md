# Степени связанности объектов

1. Наследование (доступ к `protected`)
2. Агрегация (разное время жизни)
3. Композиция (одно время жизни)
4. Дружественность (`friend`) (доступ к `private`)

## Порядок увеличения степени связанности

`2 < 3 < 1 < 4`

В идеале выбирать наименьшую степень связанности, если можно
